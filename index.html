Reachy Mini Dance Party

Voice-controlled YouTube DJ + beat-aligned dancing for your Reachy Mini.

How it works

Install this app on your Reachy Mini and just talk to it. Ask for a song, and it grabs the audio from YouTube, beat-tracks it with librosa, and choreographs head moves from the official dance library to land on every beat. Auto-DJ keeps the party going between tracks.

Try saying...

"Play some Daft Punk"
Fetches and starts a track + synced dance.
"Play something funkier"
Auto-DJ picks something complementary.
"Skip this song"
Drop the current track, advance.
"Hey Reachy, stop"
Full stop: music off, robot still, DJ idle.
"Turn it up to 80"
Volume from 0-100.
"Look at me"
Glances toward the loudest face.
"Take a photo"
Captures a still so the model can react.
"Stop dancing"
Keep the music, freeze the moves.

Tool surface

play_song fetch a track + start the party
skip_song next track
stop_party halt everything
stop_dance keep music, freeze moves
set_volume speaker volume 0-100
look_at_audience orient to largest face
look_at glance overlay
move_head manual pitch/yaw/roll
play_emotion emotion clip
set_face_tracking toggle face follow
take_photo capture and return a still

Install

The app installs through the Reachy Mini dashboard's app store — or directly via the daemon's API on your robot:

curl -s -X POST http://<robot-ip>:8000/api/apps/install \
  -H 'Content-Type: application/json' \
  -d '{"name": "reachy_mini_dance_party_app", "source_kind": "hf_space"}'

Put your OpenAI API key on the Pi (the app uses gpt-realtime):

ssh pollen@<robot-ip> 'umask 077; echo "OPENAI_API_KEY=sk-..." > ~/.env'
ssh pollen@<robot-ip> 'sudo apt-get install -y ffmpeg'
